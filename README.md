# HOW TO USE

### 特定の時間のイベントの場合

| title | startDate | endDate | startPeriod | endPeriod |
| --- | --- | --- | --- | --- |
| 任意のタイトル | yyyyMMdd | 空欄にする | 1〜7の半角数字 | 1〜7の半角数字 |
| 例 | 20250401 |  | 1 | 3 |

### 終日イベントの場合

| title | startDate | endDate | startPeriod | endPeriod |
| --- | --- | --- | --- | --- |
| 任意のタイトル | yyyyMMdd | yyyyMMdd | 空欄にする | 空欄にする |
| 例 | 20250401 | 20250402 |  |  |
